Online Mixing

Have your songs mixed by a professional mix engineer without ever having to leave your home. The process is simple. Fill out the short form below and I will then email you payment information and instructions on how to send your sessions to me. As I finish your songs, I will send you a reference mix for your review. After reviewing the reference mix you then either approve the mix, or notify me of any revisions you want (as many times as reasonably necessary).

Your satisfaction is guaranteed! If you are not satisfied with the results, you will receive a full refund.

Feel free to contact me if you have any questions.

File Prep Guidelines

The following file preparations must be properly completed in order for me to mix your songs, so please follow them or I will have to ask you to resubmit your files.

  • All files must be audio files (either .wav or .aif). MIDI files must be bounced/rendered to audio files. There is very little chance that I have the exact same MIDI set up as you, so this is necessary to get the actual sound you intended.

  • Editing must be completed before submitting for mixdown. Make sure the arrangement is correct and all tracks are clean (meaning free of clipping and unwanted distortion). Editing/arranging is available as a seperate sevice. If it is determined that a significant amount of editing is required before mixdown you will be given the option of having me do the edits at $50 per hour, or edit the material yourself and resubmit the song.

  • Each audio file will become a seperate track, so all audio files need to start at 0:00:00 on the timeline and be continuous through the end of the song. (If you send your song in a Pro Tool session you can skip this step)

  • Dry audio files (no effects) are highly recommended to allow more flexibility with effects. Remember there is no way for an engineer to remove an effect if it is already printed to the track. If you have a track with a "must-have" effect, send both wet and dry versions of the track.

  • Name all audio files with meaningful names such as kick, snare, lead vocal, etc. File names like audio1, audio2, etc. does not tell anything about what is contained in the file. This will cause a delay while I try to figure out what is on each track.

  • Create a folder with the name of the song and export your audio files to this folder. Be sure only to include files to be heard in the final mix, no alternate takes. If you are sending a Pro Tools session, be sure to send the entire project folder and not just the session file. The session file does not contain any of the actual audio!

That's it! Now you are ready to upload your music.
